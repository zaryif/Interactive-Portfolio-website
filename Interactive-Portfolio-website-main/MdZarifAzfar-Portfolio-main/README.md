
# Interactive Portfolio Website

A modern, interactive portfolio website featuring an AI-powered playground where visitors can explore real-time tools and intelligent applications. Built to showcase expertise in creating dynamic, user-focused web experiences with integrated AI capabilities.

## Features

**AI Playground**  
An interactive environment where users can engage with AI-powered tools and demonstrations, showcasing practical applications of modern language models in real-time.

**Dynamic Interactions**  
Real-time responsiveness and interactive elements that adapt to user input, creating an engaging and memorable portfolio experience.

**Responsive Architecture**  
Fully responsive design ensuring optimal viewing and interaction across desktop, tablet, and mobile devices.

**Modern Stack**  
Built with contemporary web technologies and best practices, emphasizing performance, maintainability, and scalability.

## Technical Stack

- Frontend Framework: Next.js / React
- AI Integration: Language Model API
- Styling: Tailwind CSS
- Runtime: Node.js
- Deployment: Cloud-based hosting

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your API key
3. Run the app:
   `npm run dev`
